AdiLink VIP

Private community, trusted members and global connection.

AdiLink VIP is prepared as a secure member area for people connected to Adi Naamen. The prototype below shows the structure for membership, profiles, friends, chat, events, media, moderation and admin tools.

Guest

Public access

Can view the homepage, history, culture previews and public community information.

Member

Community profile

Can create a profile, search members, send messages and participate in the community.

VIP

Exclusive access

Gets private groups, events, media archive, history content and priority community features.

Technical Setup

  • Frontend: Next.js, TypeScript, Tailwind CSS
  • Backend: Node.js and Express
  • Database: MySQL
  • Authentication: JWT and Bcrypt
  • Realtime: Socket.io
  • Map: OpenStreetMap
  • Image storage: Cloudinary later

Database Tables

users, profiles, friend_requests, friendships, messages, voice_messages, groups, group_members, events, event_participants, notifications, posts, comments, media, reports, roles

Development Phases

  1. Homepage, registration, login and profile page
  2. Friend system, direct messages and notifications
  3. Group chat, events and map
  4. Media library, history archive and moderator panel
  5. Mobile app, VIP payment and advanced features
